Built for Modern Developers

Everything you need to keep your development workflow clean and efficient.

Project Detection

Auto-detects 20+ languages and 40+ frameworks. Shows your Express, FastAPI, or Axum versions right in the table.

Docker Integration

Native mapping for Docker/Podman container IDs, images, and project names via bollard API.

Git Integration

Stop hunting for the right instance. PortForge shows you exactly which git branch is running on which port.

Health Probes

HTTP health checks with framework-aware default endpoints. Know if your service is listening but dead.

Resource Tracking

Real-time CPU and Memory usage per port, helping you track down leaked processes or heavy instances.

Orphan Cleanup

Identify zombie/orphan processes left behind by crashed IDEs and clean them safely with dry-run support.

Quick Installation

Get up and running in less than 30 seconds.

Cargo
Binary
cargo install portforge
curl -L "https://github.com/kabudu/portforge/releases/latest/download/portforge-macos-aarch64.tar.gz" | tar xz
sudo mv portforge /usr/local/bin/

Change "macos" and "aarch64" to match your system. See releases for all options.
